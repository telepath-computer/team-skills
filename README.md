# Telepath Team Skills

This repository contains skills authored by and used within the Telepath team to help us get our daily work done.

## Skills

- [Agent Supervision](agent-supervision/) — Supervise, observe, and unblock coding agents running under tmux.
- [Agent Triad](agent-triad/) — Run a supervisor, implementer, and reviewer agent group under tmux using Agent Supervision.
- [Cargo Cult Review Audit](cargo-cult-review-audit/) — Audit a system for accumulated complexity and produce ranked simplification proposals.
- [Cold Reader](cold-reader/) — Write documents for readers who have no knowledge of the history behind them.
- [Complexity Inoculation](complexity-inoculation/) — Question cascading machinery and keep solutions proportionate to their actual purpose.
- [Logical Merging](logical-merging/) — Reconcile the policies and assumptions of two branches, not only Git's textual conflicts.
- [Plain English](plain-english/) — Write clearly without private shorthand, undefined labels, or compressed jargon.
- [PR Writing](pr-writing/) — Describe a pull request's final net change for squash-merge review.
- [Protocol Alpha](protocol-alpha/) — Combine Agent Triad and a dedicated worktree to get ready for a development task.
- [Test Iteration Discipline](test-iteration-discipline/) — Choose focused test commands while iterating and broader validation only when warranted.
- [Topical PR Review](topical-pr-review/) — Organize a pull request's files and diffs into a hierarchy of logical, thematic changes.
